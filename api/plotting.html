spdcalc.plotting¶

Plotting helpers

spdcalc.plotting.plot_jsi(setup, wavelength_steps_meters, norm, /)¶

Get the Joint Spectral Intensity for coincidences over specified 2D range of wavelengths for signal/idler

Provide a SPDCSetup, Steps2D, and optional normalization if no normalization is provided, one will be calculated based on the colinear JSA

Parameters

  • setup (SPDCSetup) – The setup object

  • wavelength_steps_meters (Steps2D) – The wavelength steps object in units of meters

  • norm (float, optional) – The normalization factor. If unspecified, it will be automatically calculated.

Returns

obj:list (float) – List of values for the jsi

Get the Joint Spectral Intensity for singles over specified 2D range of wavelengths for signal/idler

Parameters

  • setup (SPDCSetup) – The setup object

  • wavelength_steps_meters (Steps2D) – The wavelength steps object in units of meters

  • norm (float, optional) – The normalization factor. If unspecified, it will be automatically calculated.

Returns

obj:list (float) – List of values for the jsi

spdcalc.plotting.calc_plot_config_for_jsi(setup, size=100, threshold=0.5, /)¶

Get the Steps2D wavelength ranges for signal/idler that centers the JSI

Parameters

  • setup (SPDCSetup) – The setup object

  • size (int, optional) – Number of steps wavelength range should have

  • threshold (float) – Value between [0., 1.]. A higher value focuses more closely on the middle of the JSI.

Returns

Steps2D – Reasonable range to view the jsi

spdcalc.plotting.calc_HOM_rate_series(setup, wavelength_steps_meters, timesteps_seconds, /)¶

Get the Hong-Ou-Mandel plot data (coincidence rate vs. time delay)

Provide a SPDCSetup, Steps2D of wavelengths, and tuple of (min, max, steps) for time delay in seconds

Parameters

  • setup (SPDCSetup) – The setup object

  • wavelength_steps_meters (Steps2D) – The wavelength steps object in units of meters

  • timesteps_seconds ((float, float, int)) – The range of timesteps to calculate over. (min, max, steps)

Returns

obj:list (float) – List of count rate values

spdcalc.plotting.calc_heralding_results(setup, wavelength_steps_meters, /)¶

Get the heralding results for a given spdc_setup setup and signal/idler wavelength range

Heralding results are a dictionary of values:

  • signal_singles_rate

  • idler_singles_rate

  • coincidences_rate

  • signal_efficiency

  • idler_efficiency

  • symmetric_efficiency

Note

rates are counts per second

Parameters

  • setup (SPDCSetup) – The setup object

  • wavelength_steps_meters (Steps2D) – The wavelength steps object in units of meters

Returns

dict – The results as specified above

spdcalc.plotting.calc_coincidences_rate_distribution(setup, wavelength_steps_meters, /)¶

Calculate the coincidence rates for each wavelength^2. Integrating over all wavelengths (all array items) will give total coincidence rate.

Parameters

  • setup (SPDCSetup) – The setup object

  • wavelength_steps_meters (Steps2D) – The wavelength steps object in units of meters

Returns

list (float) – The coincidence rates for each wavelength^2 segment

Returns

list (float) – The coincidence rates for each wavelength^2 segment
